# Dine-Time 🍽️

A fullstack restaurant booking app built with React Native + Expo + Firebase.  
Cross-platform support (iOS + Android) with guest/user booking flow, Firestore database, and dynamic slot selection.

## 🔧 Built With
- React Native + Expo
- Firebase Auth + Firestore
- Expo Router + AsyncStorage
- Formik + Yup

## 📱 Features
- User & guest login
- Book restaurants with available slots
- View booking history
- Clean and responsive UI
